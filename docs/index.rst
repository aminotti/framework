What is Yameo framework?
++++++++++++++++++++++++

Yameo framework is a set of libraries and a ready to use core application build to create multi platform sofware (HTML5, mobile and tablet) base on REST web API.

The code is open source, release under AGPL and written in python.

git clone https://github.com/aminotti/yameo.git
